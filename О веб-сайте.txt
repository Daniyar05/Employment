Чтобы посмотреть сайт в действии перейдите по ссылке: https://employment-yandex.herokuapp.com/
Простые пользователи могут добавлять, изменять и удалять свои вакансии.
Также любой пользователь(как и незарегистрированный) может перейти по ссылке (https://employment-yandex.herokuapp.com/api/users) и узнать какие люди зарегистрировались в системе.
Если перейти по: https://employment-yandex.herokuapp.com/api/users/id, где id - это id участника, то можно узнать его информацию, кроме пароля.
Так же работает и с https://employment-yandex.herokuapp.com/api/jobs и https://employment-yandex.herokuapp.com/api/users/id(О работе, которые выкладывают пользователи).

На сайте есть три типа доступа:
1) Незарегистрированные. Имеют только возможность читать вакансии.
2) Зарегистрированные. Могут добавлять, изменять и удалять свои работы.
3) Администратор(email - admin@mail.ru, password - admin). Как и 2), но может изменять и удалять вакансии других пользователей. А также может исключать самих пользователей.
Если перейти по ссылке: https://employment-yandex.herokuapp.com/user_delete, то можно посмотреть id и имена пользователей.
Если перейти по ссылке: https://employment-yandex.herokuapp.com/user_delete/id, то можно удалить пользователя с id = id 

На сайте присутствуют кнопа "На главную". Также если нажать на своё имя в правой верхней части экрана, то вы выйдете из своей учётной записи.
Для простых пользователей и администратора все функции можно проводить без использования поисковой строки.

Все возможности реализуются через функции. 
В папке data находится api и классы работы и пользователя
В папке db находится база данных пользователей и работы
В папке forms находится форма авторизации и регистрации и отправка работ
В папке static находится css файл
В папке templates находится html формы страниц
